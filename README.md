# Procurement Management & Supplier Payments System (PM-SPS)

**Система управления закупками и расчётами с поставщиками**

## Описание
Веб-приложение для автоматизации сквозного бизнес-процесса "Расчёты с поставщиками и подрядчиками" в компании IQNIX TECH.
Система позволяет управлять договорами, заявками, первичными документами, платежами и актами приёмки.

## Основной функционал (Use Cases)
1.  Управление договорами с поставщиками (CRM для поставщиков).
2.  Обработка заявок на закупку от отделов.
3.  Учёт первичных документов (счета, накладные, акты).
4.  Формирование и согласование платёжных поручений.
5.  Контроль приёмки товаров и работ.
6.  Формирование аналитических отчётов.

## Технологический стек (Tech Stack)
*   **Бэкенд:** Python (Django / FastAPI)
*   **Фронтенд:** JavaScript/TypeScript (React/Vue.js)
*   **База данных:** PostgreSQL
*   **Деплой:** Docker, Nginx

## Установка и запуск (для разработчиков)
1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/your-username/supplier-payments-system.git
    cd supplier-payments-system
    ```
2.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```
3.  Запустите сервер разработки:
    ```bash
    make dev
    # или
    python manage.py runserver
    ```
